# ECG-research
主要用于记录自己业余学习研究ECG方面的一些知识.

```某些链接需要科学上网才能打开```

> [ECG](https://zh.wikipedia.org/wiki/%E5%BF%83%E7%94%B5%E5%9B%BE):心电描记术（Electrocardiography、ECG 或者 EKG）是一种经胸腔的以时间为单位记录心脏的电生理活动，并通过皮肤上的电极捕捉并记录下来的诊疗技术。

> 某医学院校计算机相关专业毕业，有此想法源于，大学毕业设计的时候本想和小伙伴一起(没错我们专业的毕设是要组队完成的.....)研究分析心电图的，但是苦于对ECG这方面知识的匮乏和时间的紧迫性，最后不得不放弃这个方向，而且在百度查找资料的过程中不知是检索的关键词不对还是其他原因一直找不到很合适的资料（后来才知道应该多用google搜，检索英文关键词，中文的还是太少了...）。如今毕业已经快两年了，今天整理浏览器书签的时候发现了方式收藏的一些个网站，于是就想着能在业余时间学习了解这方面的知识，算是自己的一个业余兴趣爱好吧，本人英语能力较差，很多地方都是借助谷歌翻译之类的工具翻译的，其中有些错误/欠妥的翻译，请提```issue```。
# 目的（愿望）
- 熟悉了解心电图的基础知识
- 分析心电图数据
- 能否通过分析大量的数据建立一种数据模型，从而实现输入患者的一张心电图的数据，给出分析结果（神经网络，深度学习.....），例如分析患者心率是否有问题，心脏功能问题，心肌梗死，心室心房是否正常
- ......
